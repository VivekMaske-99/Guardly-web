❓ TROUBLESHOOTING:

Server won't start?
  • Check if port 3001 is in use: lsof -i :3001
  • Try different port: PORT=3002 npm start

File upload fails?
  • Check file type (PDF/DOCX/TXT only)
  • Check file size (max 50MB)
  • Check uploads directory exists

No matches detected?
  • Verify user profile data matches document
  • Check for case sensitivity (should work case-insensitive)
  • Review TESTING.md for edge cases

Processing too slow?
  • Reduce file size
  • Check system resources
  • Enable production mode: NODE_ENV=production npm start

For more help:
  • Review TESTING.md for test cases

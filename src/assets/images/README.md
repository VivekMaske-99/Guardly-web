# Images Directory

Place your UI/UX design images in this directory.

## Required Images:

### Hero Section:
- `ai-head.png` - The AI head/robot image (wireframe, glowing blue/purple)
- `shield.png` - The large wireframe shield icon
- `ui-element.png` - The floating UI element with purple border (folder icon, G1 symbol)

### Services Section:
- `security-scan.png` - Hooded figure with computer screens showing code
- `threat-defense.png` - Security professionals with control room monitors
- `privacy-compliance.png` - Shield icon with documents/data flow

## Image Requirements:
- Format: PNG (with transparency) or WebP
- Recommended size: 
  - Hero images: 800x800px or larger
  - Service images: 600x400px or larger
- Background: Transparent (PNG) or match the dark theme

## How to Add Images:
1. Export images from your UI/UX design tool (Figma, Adobe XD, etc.)
2. Name them exactly as listed above
3. Place them in this `src/assets/images/` directory
4. The components will automatically use them

If images are not found, the components will show a warning in the console but won't break.
